// Quick reference of the frontend folder layout.
# /frontend
# |- App.tsx             -> punto de entrada Expo y navegacion principal.
# |- babel.config.js     -> configuracion de Babel para React Native.
# |- tsconfig.json       -> opciones TypeScript compartidas.
# |- /src
#    |- navigation       -> navegadores y definicion de tabs/stack.
#    |- screens          -> pantallas principales de la app.
#    |- components       -> UI reutilizable (cartas, botones, etc.).
#    |- services         -> llamadas a APIs externas / backend.
#    |- types            -> tipos TypeScript compartidos.
#    |- hooks            -> hooks personalizados (estado, efectos).
